# Análise de Risco de Crédito com Lógica Fuzzy

## Visão Geral
Este projeto desenvolve um modelo avançado de avaliação de risco de crédito utilizando uma abordagem Neuro-Fuzzy (ANFIS – Sistema de Inferência Neuro-Fuzzy Adaptativo).
O modelo combina a interpretabilidade da lógica fuzzy com a capacidade de aprendizado das redes neurais, proporcionando tanto explicabilidade quanto alto desempenho preditivo.

## Contexto de Negócio
A avaliação de risco de crédito é uma função crítica nas instituições financeiras. Modelos tradicionais frequentemente enfrentam um trade-off entre acurácia e interpretabilidade.

Este projeto aborda esse desafio por meio de uma abordagem híbrida que:

- Aprende padrões a partir dos dados (semelhante a modelos de machine learning)
- Mantém regras de decisão interpretáveis (semelhante a sistemas especialistas)

O projeto integra conhecimento de domínio do setor financeiro com técnicas de análise de dados para construir um modelo realista e confiável de classificação de risco.

## Objetivo
Prever e classificar clientes com base na probabilidade de inadimplência utilizando um sistema de inferência fuzzy orientado a dados, aprimorado por aprendizado neural (ANFIS).

## Resultados / Conquistas
- Melhoria no desempenho de classificação em comparação com sistemas fuzzy básicos
- Capacidade de capturar relações não lineares entre as variáveis
- Estrutura interpretável com regras aprendidas automaticamente
