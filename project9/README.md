# Credit Risk Analysis using Fuzzy Logic

## Overview
This project develops an advanced credit risk assessment model using a Neuro-Fuzzy approach (ANFIS – Adaptive Neuro-Fuzzy Inference System). 
The model combines the interpretability of fuzzy logic with the learning capability of neural networks, providing both explainability and predictive power.

## Business Context
Credit risk evaluation is a critical function in financial institutions. Traditional models often face a trade-off between accuracy and interpretability.

This project addresses this challenge by applying a hybrid approach that:

- Learns patterns from data (like machine learning models)
- Maintains interpretable decision rules (like expert systems)

This project leverages domain knowledge from the financial sector combined with data analysis techniques to build a realistic risk classification model.

## Objective
To predict and classify clients based on their probability of default using a data-driven fuzzy inference system enhanced by neural learning (ANFIS).

## Achievements / Results
- Improved classification performance compared to basic fuzzy systems
- Ability to capture nonlinear relationships between variables
- Interpretable structure with learned rules
